# sqlmap 

## source code
- [sqlmap](https://github.com/sqlmapproject/sqlmap)

## Clone sqlmap

```sh
git clone https://github.com/sqlmapproject/sqlmap sqlmap-dev/
```
## Build image
```sh
podman build -t sqlmap -f Containerfile
```
## Run container
```sh
podman run -it --rm --name sqlmap sqlmap
```
